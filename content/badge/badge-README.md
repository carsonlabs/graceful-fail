# "Built with Graceful Fail" Badge

Add a badge to your project README to show you're using Graceful Fail for self-healing API error handling.

## Markdown

```markdown
[![Built with Graceful Fail](https://selfheal.dev/badge.svg)](https://selfheal.dev)
```

## HTML

```html
<a href="https://selfheal.dev">
  <img src="https://selfheal.dev/badge.svg" alt="Built with Graceful Fail" />
</a>
```
